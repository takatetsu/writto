# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.9.2] - 2024-12-15

### Added
- Internationalization (i18n) support: English and Japanese UI
- Ctrl+Click link navigation: internal anchors scroll to headings, external links open in default browser
- Support for plain URLs and autolinks

### Changed
- Settings dialog layout improved for better font settings display
- Removed PDF export feature (use Print instead)

## [0.9.1] - 2024-12-12

### Added
- Drag and drop file/folder moving in file explorer

## [0.9.0] - 2024-12-11

### Added
- Hanging indent for list items (bullets, numbered lists, checkboxes)
  - Text wrapping aligns with first line start position

## [0.8.1] - 2024-12-08

### Added
- Update check feature (Help > Check for Updates)
- Automatic update check on startup
- Word wrap setting persistence
- Open file from command line arguments

### Changed
- Executable name changed to `writto.exe`

### Removed
- File extension association feature (due to technical constraints)

## [0.8.0] - 2024-12-07

### Added
- Help menu and About dialog
- Japanese localization for NSIS installer
- Updated README.md (features, screenshot)

### Changed
- Unified application name to "Writto"
- Updated copyright year to 2025

## [0.7.0] - 2024-12-05

### Added
- Dark mode support
- Line number toggle (Ctrl+L)
- Word wrap toggle
- Editor width adjustment (50%-100%)
- File explorer context menu
- Draggable settings window

### Changed
- Unsaved changes warning message localized to Japanese
- Sidebar icons changed to monochrome style

## [0.6.0] - 2024-12-04

### Added
- Real-time Markdown rendering
- Syntax highlighting (multi-language support)
- Table rendering support
- Tab key indentation

## [0.5.0] - 2024-12-02

### Added
- File explorer feature
- Default startup folder setting
- Font settings (size, family)

### Fixed
- Save functionality fixes
- File switching bug fixes

## [0.1.0] - 2024-12-01

### Added
- Initial release
- Basic Markdown editor functionality
- New, Open, Save file operations
- HTML export
- Print functionality
- Search functionality
