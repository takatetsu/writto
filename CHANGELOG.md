# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.9.0] - 2024-12-11

### Added
- リスト要素のぶら下げインデント機能
  - 箇条書き、連番リスト、チェックボックスの折り返し時に2行目以降のテキスト開始位置を揃える

## [0.8.1] - 2024-12-08

### Added
- 更新確認機能を追加（ヘルプ > 更新の確認）
- 起動時の自動更新チェック
- 行の折り返し設定の永続化（再起動後も保持）
- コマンドライン引数からのファイルオープン機能

### Changed
- 実行ファイル名を `writto.exe` に変更

### Removed
- ファイル拡張子関連付け機能を削除（技術的制約のため）

## [0.8.0] - 2024-12-07

### Added
- ヘルプメニューとバージョン情報ダイアログを追加
- NSISインストーラーの日本語対応
- README.mdの更新（機能説明、スクリーンショット）

### Changed
- アプリケーション名を「Writto」に統一
- 著作権年を2025年に更新

## [0.7.0] - 2024-12-05

### Added
- ダークモード対応
- 行番号表示のトグル機能（Ctrl+L）
- 行の折り返し機能
- エディタ幅の調整機能（50%〜100%）
- ファイルエクスプローラーのコンテキストメニュー
- ドラッグ可能な設定ウィンドウ

### Changed
- 未保存変更の警告メッセージを日本語化
- サイドバーのアイコンをモノクロスタイルに変更

## [0.6.0] - 2024-12-04

### Added
- マークダウンのリアルタイムレンダリング
- シンタックスハイライト対応（多言語）
- テーブルのレンダリング対応
- Tabキーによるインデント

## [0.5.0] - 2024-12-02

### Added
- ファイルエクスプローラー機能
- 起動時のデフォルトフォルダ設定
- フォント設定（サイズ、フォントファミリ）

### Fixed
- 保存機能の修正
- ファイル切り替え時のバグ修正

## [0.1.0] - 2024-12-01

### Added
- 初期リリース
- マークダウンエディタの基本機能
- ファイルの新規作成、開く、保存
- HTMLエクスポート
- 印刷機能
- 検索機能
